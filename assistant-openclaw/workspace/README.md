# Workspace Layout

Mục tiêu: gọn, dễ tìm, dễ bảo trì.

## Cấu trúc

- `AGENTS.md` — quy tắc vận hành chính
- `SOUL.md` — persona/voice
- `USER.md` — hồ sơ làm việc với ba/mẹ
- `IDENTITY.md` — bản sắc của trợ lý
- `TOOLS.md` — ghi chú tool/local setup
- `HEARTBEAT.md` — checklist heartbeat (đang để trống)

- `cheat-sheet/` — các cheat-sheet thao tác nhanh
- `skills/` — skill cài local (ví dụ Tavily)
- `memory/` — ghi chú theo ngày `YYYY-MM-DD.md`
- `docs/` — tài liệu/rules bổ sung
- `scripts/` — script tiện ích nội bộ
- `templates/` — template dùng lại

## Quy ước đặt tên

- Cheat-sheet: `cheat-sheet/<chu-de>-quick-commands.md`
- Memory ngày: `memory/YYYY-MM-DD.md`
- Script: `scripts/<muc-dich>.sh`
- Template: `templates/<ten-template>.md`

## Ghi chú

- Không lưu secret vào workspace.
- Trước khi sửa config runtime: backup trước (khi khả thi).
- Luôn commit sau chỉnh sửa.
